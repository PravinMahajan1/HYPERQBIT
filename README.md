# DAILY/AI (AILIVE🤖) Privacy Policy Website

This package contains a complete static Privacy Policy website for the DAILY/AI (AILIVE🤖) - HyperQB Client App.

## Files

- `index.html` - Complete responsive Privacy Policy website.

## Before publishing

Open `index.html` and replace:

- `YOUR_SUPPORT_EMAIL@example.com`
- `https://YOUR-WEBSITE-URL.com`

Also review the policy and adjust it to match your final app configuration, especially:

- Analytics tools
- Crash reporting tools
- AI provider/model
- Supabase access behavior
- Account creation, if added later
- Ads, if added later
- Payments/subscriptions, if added later

## Hosting options

You can host this file on:

- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting
- Your own website/domain

After hosting, paste the live URL into Google Play Console under Privacy Policy.
